Get Your Auth0 User ID

Instructions:

  1. Open your application in another browser tab
  2. Make sure you are logged in
  3. Open the browser's developer console (F12 or right-click > Inspect > Console)
  4. Copy and paste the following code into the console:
console.log("Auth0 User ID:", JSON.parse(localStorage.getItem("auth0spa") || "{}").body?.decodedToken?.user?.sub);

This will print your Auth0 User ID in the console. Copy that ID and use it in the create-test-user.js script.

Alternative Method:

If you're already on this page and logged in to your application, click the button below to try to retrieve your Auth0 User ID:

Copy this ID and replace the placeholder in create-test-user.js
