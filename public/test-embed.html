How to Embed

  1. Open the Lutruwita map application
  2. Add routes to the map
  3. Click the "Embed" icon in the sidebar
  4. Customize the embed size if needed
  5. Copy the generated embed code
  6. Paste the code into your website's HTML

Features

  • Interactive 3D map with terrain
  • Display routes with detailed information
  • Points of Interest (POIs) with detailed information
  • Elevation profile display
  • Sidebar with route information
  • Zoom, pan, and rotate controls
  • Fullscreen mode
  • Responsive design
  • Optimized performance with caching

Implementation Improvements

  • Hierarchical data stored in Firebase for improved performance and offline capabilities
  • Optimized data structure for efficient querying and retrieval
  • Single data source for all route components (metadata, GeoJSON, POIs, etc.)
  • Simplified component architecture for better performance
  • Progressive loading for better user experience
  • Direct rendering of POIs without complex context providers
  • Automatic updates to embed data when routes are modified
  • Consistent error handling and user feedback

Firebase Implementation

The iframe above demonstrates our Firebase implementation. It uses a dedicated route data loader that loads data exclusively from Firebase, with no fallbacks to other storage systems. This approach provides a cleaner, more direct data flow and better performance.

Key aspects of the implementation:

  • Loading route metadata from Firebase's hierarchical structure
  • Loading route GeoJSON data from Firebase
  • Loading POIs from Firebase
  • Loading lines from Firebase
  • Direct error handling for Firebase-specific issues
  • Simplified data flow with no fallback complexity
